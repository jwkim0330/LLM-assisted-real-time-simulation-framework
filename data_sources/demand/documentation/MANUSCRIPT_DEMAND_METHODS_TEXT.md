# Suggested Manuscript Text for Experiment 3 Background Demand

To precisely isolate the effects of these temporal distributions, 900 controlled simulation runs were executed (three scenarios × three conditions × 100 random seeds). The core principle of this experimental design was single-variable control via paired comparisons. For each fixed scenario and random seed, the road network, initial vehicle states, and the same realization of routine background demand were held constant. The routine background demand was generated using an hourly Poisson arrival process with a total expected demand of 1,000 requests over a 24-hour period. The value of 1,000 requests was selected to create a sufficiently active background operating environment for observing the effects of the ten concentrated target requests. Because the boarding and alighting weights in the processed demand input were normalized separately to sum to 1 within each hour, all hourly intervals had the same expected rate of 41.67 requests per hour. Station-level boarding and alighting volumes obtained from the Transportation Card Big Data Integrated Information System for May 11-24, 2024 were used to determine the relative spatial probabilities of passenger origins and destinations. All 441 public-transit stops located within the geographic boundary of the study area were included. An additional 20 campus locations were assigned stakeholder-informed heuristic weights because they were not represented as public-transit stops. The processed input contains hour-specific spatial weights; because each Experiment 3 run covered 3,600 seconds from simulation time 0, the first hourly spatial-weight interval was used. Ten concentrated target requests were then injected into this fixed background environment. Across paired runs, only the temporal distribution of these ten requests varied among the Uniform, Poisson, and Synchronized conditions.
