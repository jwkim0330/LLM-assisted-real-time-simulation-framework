## Suggested PLOS ONE statement

All data underlying the findings and all author-generated code required to reproduce Experiment 3 and Figures 12-16 are available in Zenodo at https://doi.org/10.5281/zenodo.19656741. The archive includes the processed simulation inputs, archived outputs from 900 seeded simulation runs, preprocessing and statistical-analysis code, manuscript-oriented tables, and figure-generation scripts. The development repository is available at https://github.com/jwkim0330/LLM-assisted-real-time-simulation-framework.

The station-level boarding and alighting records used to construct the processed demand input were obtained from the Transportation Card Big Data Integrated Information System (TCBIS), Route and Stop Indicators, Station-level Usage Volume, for May 11-24, 2024. The exact processed `Demand.xlsx` used by the simulator, location-level provenance metadata, the documented collection and processing procedure, and an illustrative source export are included in the archive. The complete historical set of individual TCBIS stop-query files is not redistributed. Researchers seeking the original source records should access TCBIS subject to the provider's applicable access and reuse terms.

The author-generated implementation code associated with Experiment 1 is available at [EXPERIMENT 1 PUBLIC REPOSITORY AND FIXED RELEASE URL]. The author-generated implementation code associated with Experiment 2 is available at [EXPERIMENT 2 PUBLIC REPOSITORY AND FIXED RELEASE URL].
